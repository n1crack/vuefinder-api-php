VueFinder Project Template
==========================

This template provides a starting point for integrating VueFinder into
your Vue.js application.

Quick Setup
-----------

1. Install dependencies:
   npm install

2. Configure your file driver in src/main.js

3. Add VueFinder component to your app

4. Start development server:
   npm run dev

Project Structure
----------------

src/
  components/
    VueFinder.vue
  drivers/
    driver-config.js
  main.js
  App.vue

Features
--------

- Pre-configured VueFinder setup
- Example driver configurations
- Best practices included
- Ready to customize

For more information, visit the VueFinder documentation.
