# VueFinder Demo Files

Welcome to the VueFinder demo directory! This folder contains sample files to showcase the powerful features of VueFinder.

## What is VueFinder?

VueFinder is a modern, feature-rich file manager component built with Vue.js. It provides:

- **Beautiful UI**: Clean, modern interface with dark mode support
- **Full File Operations**: Upload, download, rename, delete, move, copy, and more
- **Advanced Features**: Search, preview, archive extraction, and folder navigation
- **Highly Customizable**: Extensive configuration options and theming support
- **Persistence**: Remember your last location across sessions
- **Multi-driver Support**: Works with local storage, remote APIs, and cloud services

## Folder Structure

- **Documents/**: Sample documents, PDFs, and text files
- **Images/**: High-quality stock photography samples
- **Media/**: Audio and video files for testing
- **Projects/**: Sample project files and templates
- **Code/**: Code snippets and examples
